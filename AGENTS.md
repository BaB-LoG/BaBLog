# Repository Guidelines

## 필수 컨텍스트 로딩 지침
- 이 디렉토리에서 작업을 시작할 때 아래 문서를 먼저 읽고 맥락을 확보하세요.
  - `context.md`
  - `BaBLog-FE/AGENTS.md`
  - `BaBLog-BE/AGENTS.md`
- 위 문서들은 프론트/백엔드 구조, API, 인증 흐름, 주요 구현 포인트를 포함합니다.

## 응답 언어
- 모든 답변은 한국어로 작성합니다.

## 작업 범위 요약
- 이 저장소는 BaBLog 프론트엔드(Vue 3 + Vite)와 백엔드(Spring Boot + MyBatis)를 함께 포함합니다.
- 구체적인 구현 질문(기능, API, 데이터 흐름, 컴포넌트 동작 등)에 답하기 위해 `context.md`의 상세 요약을 참고합니다.
